# Goal

## Project
textblob - a Python text processing library

## Description
A simple Python library for processing textual data. It provides a straightforward API for common natural language processing (NLP) tasks including:
- Part-of-speech (POS) tagging
- Noun phrase extraction
- Sentiment analysis
- Text classification (Naive Bayes, Decision Tree)
- Tokenization (splitting text into words and sentences)
- Word and phrase frequencies
- N-grams
- Word inflection (pluralization and singularization)
- Lemmatization and stemming
- Spelling correction
- WordNet integration

The library wraps NLTK to provide a more accessible interface for common text processing tasks.

## Scope
- 10-15 production source files to implement
- 10-12 test files to write
- Core modules: main text classes, tokenizers, taggers, sentiment analyzers, noun phrase extractors, classifiers, parsers
